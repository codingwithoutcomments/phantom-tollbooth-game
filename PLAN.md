# The Phantom Tollbooth Game - Build Plan

## Game Flow
```
Title Screen → Book Summary → Character Select
  → Driving Scene → "Welcome to the Lands Beyond!"
  → The Doldrums (tap to think!)
  → Drive into Dictionopolis (sign)
  → Level 1: Word Market (match synonyms)
  → Driving onward...
  → Bonus: Chroma's Orchestra (color/rhythm mini-game)
  → Drive into Digitopolis (sign)
  → Level 2: Number Mine (addition & subtraction)
  → Mountains of Ignorance transition
  → Level 3: Demon Dodge Staircase (platformer to rescue Rhyme & Reason)
  → Victory: Rescue Rhyme & Reason!
  → End screen with book theme message
```
